# SportsAnalytics Pro - PWA

A standalone, installable web app for sports betting analytics.

## Features
- Install on iOS/Android/Home Screen
- Works offline
- Local storage for bankroll & bets
- AI Value Bet calculations
- Poisson distribution models
- Real-time odds comparison

## Installation

1. **Desktop**: Open `index.html` in any modern browser
2. **Mobile**: 
   - iOS: Open in Safari → Share → "Add to Home Screen"
   - Android: Open in Chrome → Menu → "Install App"

## Usage
No build process required! Just double-click `index.html` or host on any static server.

## Hosting Options
- GitHub Pages (free)
- Netlify Drop (free)
- Vercel (free)
- Any web server (Apache, Nginx, etc.)
